# Investment Forecast Android Web Wrapper

Готов Android Studio проект (Kotlin + Compose), който зарежда уеб версията на приложението в WebView.

## Как работи
- По подразбиране отваря: https://bamze1970.github.io/investment-forecast-sync-v3/
- От бутона за настройки в приложението можеш да смениш URL към по-нова версия, например:
  https://bamze1970.github.io/investment-forecast-sync-v4/

## Защо този подход
- Уеб версията вече има sync, Firebase и бизнес логика.
- Android приложението дава инсталируем shell с икона, back navigation, refresh и настройки.

## Как да стартираш
1. Отвори проекта в Android Studio Ladybug или по-нова.
2. Изчакай Gradle sync.
3. Стартирай на емулатор или телефон.

## Забележка
В тази среда не мога да произведа подписан APK/AAB, но проектът е подготвен за директен import в Android Studio.
